# Roadmap

## Current Status

**Phase 1 — CSE Foundation** ✅ Working

Tracium Engine executes Java code via JDI, captures full state at every step, produces UEF traces. Nerva persists traces to disk.

## Vision

Tracium aims to become execution intelligence infrastructure — the foundation layer that any tool can build on. Production recording, execution diffing, distributed correlation, and AI reasoning are all on the path.

# Prism

Prism is the visualization platform. It renders UEF traces as interactive step-by-step playbacks and architecture graphs as navigable topology views.

## Surfaces

- **Runtime Studio** — UEF trace playback with code viewer, stack/heap panels
- **Architecture Lens** — UGF topology with service graph and dependency view

## Lenses

- **Debug Truth** — raw state, precise frames, exact references
- **Learning Lens** — same data, calmer explanation

## Status

Prism has a working UI with two surfaces and two lenses. Wiring to Nerva for real trace data is in progress.

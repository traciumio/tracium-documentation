# Pulse <Badge type="warning" text="Planned" />

Pulse brings Tracium into the developer's editor. Trace code, explore architecture, and navigate between execution and source without leaving your IDE.

## Planned Hosts

- **VS Code** — first implementation
- **JetBrains** — second implementation (IntelliJ, WebStorm)

## Planned Workflows

### Runtime
Select code → run with Tracium → see step-by-step trace inline with variable values on hover.

### Architecture
Analyze project → see dependency graph in webview → click nodes to navigate to source.

## Tech Stack

| Host | Technology |
|------|-----------|
| VS Code | TypeScript, Extension API |
| JetBrains | Kotlin, IntelliJ Platform SDK |

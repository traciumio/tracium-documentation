Quanta
Shared Specs & Contracts
Canonical schemas and contracts that all products share. Defines UEF (Universal Execution Format) and UGF (Unified Graph Format) — the common language of the ecosystem.
JSON Schema UEF UGF Markdown

T
Tracium Engine
CSE — Code State Engine
Headless runtime execution engine. Executes code, captures state through JDI, normalizes into timeline, and emits UEF. The core intelligence that powers runtime understanding.
Java 21 JDI Gradle Headless
SERVICE
A
Atlas
Repository Analyzer
Static code intelligence engine. Ingests repositories, builds symbol tables, resolves dependencies, constructs call graphs, and emits UGF for architecture visualization.
Java 21 Gradle AST Parsing Graph Analysis
SERVICE
N
Nerva
Orchestration Layer
Platform backbone. Manages sessions, authentication, GitHub integration, and routes requests between clients and backend engines. The API gateway for the ecosystem.
TypeScript REST API OAuth Sessions
APP

Prism
Visualization Platform
The user-facing product. Renders runtime traces as interactive step-by-step playbacks and repository structures as navigable architecture graphs. Where intelligence becomes visual.
React TypeScript Vite Interactive UI

Developer-facing TypeScript SDK. Typed client for runtime tracing, repository analysis, and GitHub integration. Enables external tools to build on the Tracium ecosystem.

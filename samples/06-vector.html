SDK

Vector

TypeScript SDK for the Tracium ecosystem. Typed client for runtime tracing, repository analysis, and programmatic access to code intelligence.

Installation

$ npm install @tracium/vector

Quick Start

TypeScript Basic Runtime Trace
import { TraciumClient } from '@tracium/vector';

// Initialize client
const client = new TraciumClient({
  baseUrl: 'http://localhost:3000',
  apiKey: 'tk_your_api_key'
});

// Submit Java code for runtime tracing
const session = await client.runtime.create({
  language: 'java',
  entrypoint: 'Main.main',
  code: `class Main {
    public static void main(String[] args) {
      int[] arr = {5, 3, 1};
      for (int i = 0; i < arr.length; i++) {
        System.out.println(arr[i]);
      }
    }
  }`,
  limits: { timeoutMs: 5000, maxSteps: 500 }
});

// Wait for execution to complete
await client.runtime.waitForCompletion(session.id);

// Get the full trace
const trace = await client.runtime.trace(session.id);

console.log(`Total steps: ${trace.steps.length}`);

// Navigate through each step
for (const step of trace.steps) {
  console.log(
    `Step ${step.step}: ${step.event} at line ${step.source.line}`
  );
}
TypeScript Repository Analysis
// Analyze a local repository
const analysis = await client.repo.create({
  language: 'java',
  source: {
    type: 'local',
    rootPath: '/workspace/my-project'
  },
  analysisMode: 'architecture'
});

await client.repo.waitForCompletion(analysis.id);

// Get the architecture graph
const graph = await client.repo.graph(analysis.id);

// Explore nodes and edges
for (const node of graph.nodes) {
  console.log(`${node.type}: ${node.name}`);
}

// Find most-depended-on classes
const deps = graph.edges
  .filter(e => e.type === 'DEPENDS_ON')
  .reduce((acc, e) => {
    acc[e.target] = (acc[e.target] || 0) + 1;
    return acc;
  }, {});

SDK Modules

client.runtime

Runtime execution and trace access

.create(request) → Session
.get(id) → SessionStatus
.trace(id) → UEFTrace
.steps(id, opts) → Step[]
.step(id, n) → Step
.state(id, n) → State
.waitForCompletion(id) → void

client.repo

Repository analysis and graph access

.create(request) → Session
.get(id) → SessionStatus
.graph(id) → UGFGraph
.symbols(id) → Symbol[]
.waitForCompletion(id) → void

client.github

GitHub integration

.connect() → AuthURL
.repos() → Repository[]
.disconnect() → void

client (platform)

Health and capability discovery

.health() → HealthStatus
.capabilities() → Capabilities
.schemas.uef() → JSONSchema
.schemas.ugf() → JSONSchema

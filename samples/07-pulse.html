Key Features

Inline Value Decorations

See variable values right next to the code as you step through execution. Values update in real-time as you navigate the trace.

Trace Sidebar

Full trace step list, call stack, and variable inspector in the VS Code sidebar. Click any step to jump to that point in execution.

Architecture Webview

Interactive dependency graph rendered in a VS Code webview panel. Click nodes to navigate directly to source files.

Context Menu Integration

Right-click any file or selection to trace it. Seamless integration with your existing VS Code workflow.

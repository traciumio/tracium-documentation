Key Features

Step-by-Step Playback

Navigate forward and backward through execution. Every step shows the exact state of stack, heap, and variables.

Truth-First State

Object identity, aliasing, and null values are shown exactly as the JVM sees them. No simplification that hides the truth.

Delta Highlighting

Every step highlights what changed. New values, updated array elements, and modified fields are instantly visible.

Dual Mode

Learning mode for clarity and minimal cognitive overhead. Debug mode for full state accuracy and execution truth.

API Endpoints

Runtime Analysis

POST /v1/sessions/runtime Create runtime trace session
Request Body
{
  "language": "java",
  "entrypoint": "Main.main",
  "code": "class Main {\n  public static void main(String[] args) {\n    int x = 5;\n  }\n}",
  "limits": {
    "timeoutMs": 5000,
    "maxSteps": 1000
  }
}
Response
{
  "sessionId": "sess_rt_001",
  "status": "PENDING",
  "createdAt": "2026-04-04T10:15:00Z"
}
GET /v1/sessions/runtime/{id} Get session status
Response
{
  "sessionId": "sess_rt_001",
  "status": "COMPLETE",
  "totalSteps": 47,
  "completedAt": "2026-04-04T10:15:03Z"
}
GET /v1/sessions/runtime/{id}/trace Get full UEF trace
Response: Full UEF document
// Returns complete UEF JSON with all steps, states, and deltas

Repository Analysis

POST /v1/sessions/repo Create repository analysis session
Request Body
{
  "language": "java",
  "source": {
    "type": "github",
    "owner": "myorg",
    "repo": "todo-app",
    "branch": "main"
  },
  "analysisMode": "architecture"
}
GET /v1/sessions/repo/{id}/graph Get UGF graph
Response: Full UGF document
// Returns complete UGF JSON with nodes, edges, and metadata

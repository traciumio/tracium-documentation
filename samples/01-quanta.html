Quanta

The shared contracts and schemas that unify the entire Tracium ecosystem. Every product speaks the same language through UEF and UGF.

What Quanta Defines

UEF - Universal Execution Format

  • Session model (language, adapter, entrypoint)
  • Step model (event, source anchor, delta, state)
  • State model (frames, heap, globals, output)
  • Value kinds (primitive, reference, object, array)
  • Delta model (frame changes, heap changes)
  • 14 core event types
  • Diagnostics model

UGF - Unified Graph Format

  • Node types (repo, module, package, class, method)
  • Edge types (DECLARES, IMPORTS, CALLS, etc.)
  • Certainty levels (high, medium, low)
  • Source anchor references
  • Architecture groupings
  • Graph metadata
